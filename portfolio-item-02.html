welcome to myso_longproject

2D game with minlibx

This project is a small 2D game with minilibx. NOTE !!! The MiniLibX! is a library that was developed internally and includes basic necessary tools to open a window, create images and deal with keyboard and mouse events.

Foreword

Being a developer is a great thing for creating your own game.

But a good game needs some good assets. In order to create 2D games, you will have to search for tiles, tilesets, sprites, and sprite sheets.

Fortunately, some talented artists are willing to share their works on platforms like: itch.io

In any case, try to respect other people’s work.

objectives

It’s time to create a basic computer graphics project!

so long helped me to improve my skills in the following areas: window management, event handling, colors, textures, and so forth.

I used the MiniLibX graphical library! This library was developed internally and includes basic necessary tools to open a window, create images and deal with keyboard and mouse events.

The other goals are: being rigorous, level up in C programming, use basic algorithms, do some information research, and so forth.

common information

  • The project is written in C.
  • The project is written in accordance with the Norm.
  • Functions should not quit unexpectedly (segmentation fault, bus error, double free, etc) apart from undefined behaviors.
  • All heap allocated memory space is properly freed when necessary.
  • The project include a Makefile which will compile source files to the required output with the flags -Wall, -Wextra and -Werror, using cc, and the Makefile doesn't relink.
  • Makefile contain the rules $(NAME), all, clean, fclean and re.
